請針對每個活動場次頁面，進行以下調整：

## 一、刪除
- 移除「賓客名單」區塊（整個區塊含新增按鈕一併移除）

## 二、保留並複製擴充
### 現場人員班表 → 分為「外場人員」與「內場人員」兩個獨立區塊
每個人員區塊格式：
- 欄位：姓名、職務、組別（外場/內場）、到場時間、離場時間、備註
- 可新增/刪除人員列
- 底部統計：外場人數 X 人 / 內場人數 X 人

### 任務清單 → 分為「外場任務」與「內場任務」兩個獨立區塊
每個任務區塊格式：
- 欄位：任務名稱、負責人、時間點、組別（外場/內場）、完成狀態（checkbox）
- 可新增/刪除任務列

## 三、新增區塊（依以下順序排列在頁面中）

### 1. 前一天準備清單
- checkbox 樣式，可勾選完成
- 欄位：準備項目、負責人、數量/說明、完成狀態
- 預設填入幾個常見項目（如：確認場地電源、備品清點、菜單列印）
- 可新增/刪除項目

### 2. 桌位配置
- 上方可上傳一張桌位圖片（接受 jpg/png，儲存為 base64 至 localStorage）
- 圖片顯示區域下方有「更換圖片」按鈕
- 圖片旁邊或下方有備註欄（自由輸入桌次說明）

### 3. 圓桌/IBM桌擺放位置
- 獨立區塊，標題：「圓桌 / IBM桌擺放說明」
- 表格格式：桌號、桌型（圓桌/IBM桌/長桌/吧台）、位置說明、人數、備註
- 可新增/刪除列
- 底部自動統計各桌型數量

### 4. 每位現場人員的當日 Schedule
- 標題：「人員當日時間表」
- 從「現場人員」（外場+內場）資料自動帶出人員姓名
- 每位人員顯示一個小時間表區塊：
  - 人員姓名（加上職務標籤）
  - 可新增時間點 + 任務描述的列
  - 可刪除列
- 若人員班表新增人員，此區塊也自動產生對應的個人時間表
- 每人時間表可折疊/展開（預設展開）

## 四、資料同步
- 所有新增區塊的資料都要納入 GAS_URL 同步（pushToSheets / fetchFromSheets）
- 圖片（base64）只存 localStorage，不同步到 Google Sheets（避免超出大小限制）
- 其餘欄位正常同步

## 五、版面順序（由上至下）
1. 活動基本資訊
2. 前一天準備清單  ← 新增
3. 菜單清單
4. 時程表
5. 外場人員 / 內場人員
6. 人員當日時間表  ← 新增
7. 外場任務清單 / 內場任務清單
8. 桌位配置（含圖片上傳）  ← 新增
9. 圓桌/IBM桌擺放位置  ← 新增
10. 預算明細
11. 備註 / 特殊需求
