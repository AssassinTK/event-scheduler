# 🎉 活動排程管理系統

一個簡單易用的活動管理工具，特別為婚禮、宴會、派對等大型活動設計。

**線上體驗：** (部署後會有網址)

---

## ✨ 核心功能

### 📅 活動管理
- 建立和編輯多個活動
- 自動同步到 Google Sheets
- 查看活動總覽和詳細資訊

### 👥 人員班表
- 外場 / 內場人員管理
- 出勤狀態追蹤（確認/不出勤）
- 到離場時間設定
- 人員庫快速添加

### ✅ 任務清單
- 外場/內場任務分類
- 指派給具體人員
- 任務完成狀態追蹤
- 時間排序

### 📊 視覺化管理
- **甘特圖**：一眼看清人員和活動時間線
- **個人行程表**：每位員工的當日排程
- **樹狀折疊**：快速瀏覽或詳細編輯

### 🖨️ 列印功能
- 完整活動詳情列印
- **個人行程單獨列印**（✨ 新功能）
- 打印格式自動優化

### 💾 數據管理
- **本機保存**：自動存到瀏覽器
- **Google Sheets 同步**：備份到雲端
- **自動同步**：每 30 秒自動拉取最新資料
- **JSON 匯出**：完整備份

### 🎨 使用體驗
- 深色模式支援
- 字體大小調整
- 響應式設計（電腦/平板都適用）

## 💡 進階功能

### 自動同步
```
設定 → Google Sheets 同步設定 → 貼入 GAS URL → 啟用 🔄 自動同步
```
啟用後，每 30 秒自動拉取 Google Sheets 最新資料，實現多人實時協作。

### 個人列印
打開人員時間表 → 點擊員工名稱 → 🖨️ 列印按鈕
生成該員工個人行程表，可直接列印或另存 PDF。

### 批量操作
在人員庫中：
- 「全選 О」：快速設定所有員工出勤
- 「📋 複製時間」：從一場次複製時間到另一場次

---

## 🐛 故障排查

### 資料丟失
→ 在設定頁點「📥 從 Sheets 匯入」恢復

### 同步失敗
→ 檢查 GAS URL 是否正確（包括 `/exec` 結尾）
→ 確認 GAS 訪問權限是「所有人」

### 頁面閃爍
→ 刷新瀏覽器
→ 清除快取（Ctrl+Shift+Delete）
