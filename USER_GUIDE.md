# 📖 使用指南：活動排程管理系統

這是給使用者（不只是開發者）的完整使用說明。

---

## 🎯 第一次使用？跟著這個走

### 1️⃣ 打開 App

你會看到這樣的界面：
```
┌─────────────────────────────────────┐
│  📅 總覽  | 🎊 3/7晚宴 | 💍 3/8午宴 | 👥 全員 | ⚙️ 設定
└─────────────────────────────────────┘
```

### 2️⃣ 設定數據源（一次性）

第一次用，需要連接 Google Sheets 備份和團隊共享：

1. 點擊右上角 **⚙️ 設定**
2. 找到「Google Sheets 同步設定」區塊
3. 貼入管理員給你的 **GAS Web App URL**
4. 點擊「🔗 儲存 URL」
5. **可選**：開啟「🔄 自動同步」開關
   - 這樣每 30 秒會自動從 Google Sheets 拉取最新資料
   - 適合團隊協作時使用

✅ 完成！現在可以開始用了。

## 📅 頁面說明

### 「📅 總覽」頁面

顯示所有活動的概覽。

**活動卡片**
```
┌─────────────────────────┐
│  🎊 3/7晚宴    [進行中] │
│  📅 2026-03-07          │
│  🕐 17:00 - 21:00       │
│  👥 130人               │
└─────────────────────────┘
```
點擊卡片進入詳細編輯。

**活動列表**
- 日期、時段、名稱、狀態都可以直接編輯
- 右側「🗑️」刪除場次
- 下方「+ 新增場次」建立新活動

**前置準備清單**
- 活動共通的準備事項
- 例如「3/6 喜餅送達」、「3/7 新人入住」

---

### 活動詳情頁（例如「3/7晚宴」）

進入活動後，會看到完整的編輯界面。

#### 📋 1. 基本資訊卡片
```
📍 信義區 A Beach 101        🕐 17:00-21:00
👥 130人                     💰 NT$150,000
```
點擊直接編輯。

#### ⏰ 2. 時程表
活動的流程時間線
```
時間    內容
14:30   新娘化妝（員工休息室，不可進出）
16:30   戶外佈置，水杯130個，零食湯碗50個
18:30   🎙開場誓詞（18:30-18:50不可走動）
```

**操作**
- 點擊時間或內容直接編輯
- 右側「❌」刪除此項
- 下方「+ 新增時程」添加新項目

#### 👔 3. 外場人員

所有外場工作人員的班表。

```
姓名        崗位        到場時間   離場時間   出勤   備註
張良瑋      外場領班    16:00      22:00      O      全場督導
外場組長    外場服務員  16:30      21:30      O
```

**操作**
- **直接編輯**：點擊任何欄位改資料
- **出勤狀態**：點擊出勤欄位切換
  - 「－」= 未確認
  - 「O」= 出勤
  - 「X」= 不出勤
- **刪除**：右側「❌」
- **新增**：右上「+ 新增外場人員」或下方快速新增框

**快速新增人員**
下方有快速新增框（虛線邊框）：
```
[從人員庫選 / 手動輸入] [姓名] [崗位▼] [外場▼] [10:00→22:00] [負責區域]
```
- 「從人員庫選」：列出已有的員工
- 「手動輸入」：直接輸入新人名字

#### 👨‍🍳 4. 內場人員

同外場人員，但用於廚房/後台工作人員。

#### 📅 5. 人員當日時間表

每個員工詳細的當日排程。

```
▼ 張良瑋 | 外場領班 · 外場

時間    任務描述
16:00   抵達現場，全場檢查
17:00   迎賓準備
18:00   協助引導賓客
```

**操作**
- 點擊時間/任務編輯
- 「📋 從任務帶入」：自動把任務清單中指派給他的任務加到這裡
- **🖨️ 列印**：只列印這個人的行程表（新功能！）

#### ✅ 6. 外場任務 / 內場任務

待辦事項清單，可指派給特定人員。

```
完成   任務名稱            負責人    時間點
□      場地佈置確認        張良瑋    16:00
☑      水杯備齊130個        外場組    16:30
□      素食餐點確認        廚房      17:00
```

**操作**
- **完成狀態**：點擊左邊方框勾選完成
- **直接編輯**：點擊任何欄位改資料
- **刪除**：右側「❌」
- **新增**：下方快速新增框或右上「+ 新增」

#### 🍱 7. 菜單清單

食物品項和份數。

```
照片   品項名稱    份數   備註    內場確認
📷    爐烤牛排    73     新鮮   ✅ 已確認
📷    肋眼豬排    23     全熟   ⏳ 未確認
```

**操作**
- **上傳照片**：點擊 📷 區域上傳菜色照片
- **直接編輯**：點擊欄位編輯
- **內場確認**：點擊「已確認」/「未確認」按鈕

#### 💰 8. 預算明細

估算 vs 實際支出。

```
項目名稱     預估金額    實際金額   備註
場地費       50,000      50,000    已付訂金
佈置費用     15,000      16,500
餐點費用     85,000      83,500

總計         150,000     150,000   差異：0 持平
```

顏色提示：
- 🟢 結餘（實際 < 預估）
- 🔴 超支（實際 > 預估）
- ⚪ 持平

#### 🗺️ 9. 桌位配置

上傳活動現場的座位圖，可加文字說明。

點擊上傳圖片，然後添加說明文字。

#### 🪑 10. 圓桌/IBM桌擺放說明

```
桌號   桌型      位置說明   人數   備註
A1     圓桌      靠窗區     10     主桌
B1     IBM桌     中央區     8      IBM團隊
C1     長桌      後方       12     親友團
```

詳細說明各桌位配置。

#### 📝 11. 備註/特殊需求

自由編寫活動備註。

例如：
```
🥗 蛋奶素1位坐L7
🪑 主桌G3，IBM桌×3
👷 工作人員餐盒9份
新娘對花卉過敏，避免使用百合
主桌需要特別準備無酒精飲料
```

#### 📊 12. 甘特圖（最下方）

視覺化時間線。顯示：
- 活動流程（藍色）
- 人員排班（綠色=外場，橙色=內場）
- 任務時間點（小方塊）

可以 🔍− 縮小 / 🔍+ 放大 來調整粒度。

### 👥 「全員」頁面

所有員工的統一班表，跨多個場次。

**人員庫表格**

```
照片   姓名      組別    備註    3/7晚宴   3/8午宴   3/15午宴
📷    張良瑋    外場    店長    ✅ O      ✅ O      ✅ O
                              10:00-    09:30-    10:00-
                              22:00     15:00     15:00
```

**操作**
- **出勤狀態**：點擊「O」/「X」/「-」切換
- **時間選擇**：狀態為「O」時，點擊下面的時間選擇器設定到離場時間
- **新增員工**：下方「+ 新增員工」按鈕
- **全選 O**：快速把某場次所有員工設為出勤
- **📋 複製時間**：從一場次的時間複製到另一場次

**任務清單**（下方）

全公司共通的任務：
- 吧台：酒水杯具數量確認
- 外場：130個水杯準備
- 廚房：食材備貨檢查

點擊左邊方框標記完成。

### ⚙️ 「設定」頁面

#### ① Google Sheets 同步設定

**保存 GAS URL**
貼入 Web App URL 後點「🔗 儲存 URL」。

**即時同步**
點「☁️ 即時同步」立即推送資料到 Google Sheets。

**自動同步**
開啟後會每 30 秒自動拉取 Google Sheets 最新資料。

#### ② 資料匯入

點「📥 從 Sheets 匯入」會把 Google Sheets 最新資料覆蓋到本機。

⚠️ 注意：會覆蓋目前的本機資料，請先確認。

#### ③ 資料管理

**💾 匯出 JSON**
把所有資料下載為 JSON 檔案，可用於備份或轉移到其他電腦。

**🗑️ 清除所有資料**
完全清除所有資料，無法復原。請先確認。

## 💾 保存和同步

### 自動保存（本機）
所有編輯都自動保存到瀏覽器的本機存儲。
- 關閉瀏覽器重新開啟也不會丟資料
- 清除快取會丟資料，所以定期備份很重要

### 同步到 Google Sheets
1. **手動同步**：點設定頁「☁️ 即時同步」
2. **自動同步**：開啟自動同步後每 30 秒同步一次

優點：
- 所有人看同一份資料
- Google Drive 自動保留版本歷史
- 可以分享給特定人員看

### 備份
定期點設定頁「💾 匯出 JSON」下載備份。

## 🐛 常見問題

### Q: 資料丟失了怎麼辦？
**A:**
1. 檢查 Google Sheet 是否有備份
2. 點設定「📥 從 Sheets 匯入」恢復
3. 如果也沒有，檢查瀏覽器歷史記錄找 JSON 備份

### Q: 為什麼和團隊的資料不一致？
**A:**
1. 檢查自動同步是否開啟
2. 點「☁️ 即時同步」強制同步
3. 等待自動同步（30 秒）

### Q: 可以在手機上用嗎？
**A:** 可以，但建議用電腦。手機上：
- 編輯會比較擁擠
- 部分表格可能需要滾動
- 列印功能用電腦更好

### Q: 刪除員工會怎樣？
**A:**
1. 從「全員」頁刪除會移除整個員工
2. 從活動詳情刪除只移除該活動的員工
3. 都無法復原，請先備份

### Q: 怎麼修改過去的活動？
**A:**
1. 點「📅 總覽」找到該活動
2. 點活動卡片或表格列進入
3. 編輯完成自動保存

## 🎯 進階技巧

### 快速複製人員班表
在「全員」頁面：
1. 進行第一場次的人員班表（姓名、時間等）
2. 點「📋 複製時間」選擇目標場次
3. 自動複製時間到其他場次

### 快速導入任務到個人時間表
在人員時間表：
1. 展開員工
2. 點「📋 從任務帶入」
3. 自動把該員工的任務加到時間表

### 批量更新出勤狀態
在「全員」頁：
1. 點「全選 O」快速把某場次所有人設為出勤
2. 手動調整不出勤的人

### 跨場次時間對齐
如果多場活動時間類似：
1. 在「全員」頁用「📋 複製時間」
2. 自動同步員工到場和離場時間
